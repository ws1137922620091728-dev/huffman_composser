# 基本用法

## 压缩文件

```
python huffman_compressor.py compress <输入文件> [输出文件]
```

示例：

```
python huffman_compressor.py compress test.txt test.huff
```

## 解压文件

```
python huffman_compressor.py decompress <输入文件> [输出文件]
```

示例：

```
python huffman_compressor.py decompress test.huff test_decompressed.txt
```
